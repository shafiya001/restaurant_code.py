# Restaurant Reservation System

This project is a simple web application built using **Python** and **Streamlit** that allows users to make table reservations at different restaurants. It demonstrates how to build an interactive website with form inputs, session storage, and image display using Streamlit.

---

## Features

- Easy-to-use interface for selecting restaurants  
- Option to choose reservation date and time  
- Ability to specify the number of guests  
- Displays confirmed reservations in the current session  
- Shows restaurant details and images  
- Fully built with Streamlit (no external backend)

---

## Tech Stack

- **Language:** Python  
- **Framework:** Streamlit  
- **Data Handling:** Streamlit Session State
